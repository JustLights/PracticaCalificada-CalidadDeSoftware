# PracticaCalificada-CalidadDeSoftware
El repositorio detalla la ejecución de la práctica sobre niveles de prueba de software, enfocada en una API REST de gestión de tareas (To- Do list) desarrollada en Node.js con Express y PostgreSQL. La práctica aborda el diseño y la implementación de pruebas unitarias, de integración y end-to-end.
